Clasificación de Cáncer de Mama con Regresión Logística

Introducción

La predicción temprana de tumores malignos es crucial para mejorar los protocolos de diagnóstico y tratamiento oncológico. Utilizando ciencia de datos, analizamos el conjunto de datos de cáncer de mama de Wisconsin, construyendo un modelo de regresión logística que predice si el tumor es maligno o benigno.

El proyecto valida la eficacia de la regresión logística tras un cuidadoso preprocesamiento de datos y destaca el uso de Pandas, Seaborn y Scikit-learn para análisis y modelado.

Código y notebook disponibles en GitHub.

Limpieza y Preparación de Datos

Se exploró y cargó el dataset con Pandas. Los valores faltantes se imputaron o descartaron según el contexto. Las variables categóricas (por ejemplo, diagnóstico) se codificaron mediante label encoding. Se normalizaron las características numéricas para facilitar la convergencia del modelo.

El Análisis Exploratorio de Datos (EDA) con Seaborn permitió identificar correlaciones entre características morfológicas y la malignidad del tumor.

Herramientas y Librerías
  • Python: Lenguaje principal.
  • Pandas & NumPy: Limpieza y manipulación de datos.
  • Seaborn & Matplotlib: Visualización y análisis de correlaciones.
  • Scikit-learn: Modelado y validación.
Modelos de Machine Learning

Se compararon múltiples algoritmos, destacándose:

  • Regresión Logística: Modelo base y altamente interpretable.
  • SVM: Captura relaciones no lineales.
  • Random Forest: Mejora la precisión y reduce el sobreajuste.

La validación cruzada y la métrica ROC-AUC demostraron que la regresión logística ofrece un rendimiento competitivo con gran interpretabilidad.

Flujo de Trabajo
  1. Carga de Datos: Importar y explorar el dataset.
  2. Preprocesamiento: Imputación, normalización y codificación.
  3. Análisis Exploratorio: Visualizar correlaciones clave.
  4. Ingeniería de Características: Seleccionar variables relevantes.
  5. Modelado: Entrenar y evaluar modelos.
  6. Interpretación: Analizar coeficientes y métricas.

Gráfico de importancia de características para el modelo de regresión logística.

Resultados
  • Precisión superior al 97 % utilizando regresión logística.
  • Alta interpretabilidad a través de los coeficientes del modelo.
  • SVM alcanzó ROC-AUC marginalmente mayor, pero con menor interpretabilidad.
  • El pipeline reproducible facilita su integración en flujos clínicos.
